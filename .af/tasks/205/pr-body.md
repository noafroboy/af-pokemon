## Summary
- Implements the complete audio system for the Pokemon game: Tone.js chiptune music sequencer (9 tracks), Howler.js SFX playback (14 WAV files), and Web Audio API FM-synthesized Pokemon cries (21 species).
- All audio operations are wrapped in try/catch with SSR guards, ensuring the game remains fully playable when audio is blocked or unavailable.
- SFX WAV files are generated by a pure Node.js synthesis script (`npm run generate-sfx`) with no external audio dependencies.

## Changes

### New Files
- `src/game/audio/cry-params.ts` — `CRY_PARAMS` map for 21 Pokemon species (FM synthesis params), `synthesizeCry()` function using Web Audio oscillator + ADSR envelope
- `src/game/audio/tracks/overworld-tracks.ts` — Note data for TITLE_THEME, PALLET_TOWN, ROUTE_1, POKEMON_CENTER, VICTORY_FANFARE, BADGE_FANFARE
- `src/game/audio/tracks/battle-tracks.ts` — Note data for WILD_BATTLE, TRAINER_BATTLE, GYM_BATTLE
- `src/game/audio/chiptune-tracks.ts` — `CHIPTUNE_TRACKS` Record barrel export with `NoteEvent`/`TrackDefinition` types
- `src/game/audio/sfx-sprites.ts` — `SFX_FILES` mapping 14 SFX IDs to `/assets/sfx/*.wav` paths
- `src/game/engine/AudioManager.ts` — Singleton AudioManager with lazy Tone.js + Howler.js init, `playMusic`, `stopMusic`, `playSFX`, `playTextBlip`, `playCry`, `setMusicVolume`, `setSFXVolume`, `onFirstGesture`
- `scripts/generate-sfx.mjs` — Node.js WAV synthesis script generating all 14 SFX files into `public/assets/sfx/`
- `public/assets/sfx/*.wav` — 14 generated SFX WAV files

### Modified Files
- `package.json` — Added `"generate-sfx"` npm script
- `src/game/engine/InputManager.ts` — Added `setFirstGestureCallback()` + fires on first keydown to unlock Web Audio API
- `src/game/engine/GameEngine.ts` — Integrated AudioManager: title music on start, map music on load, encounter-flash SFX, wild/trainer battle music + cry on battle start, badge fanfare on BADGE_CEREMONY, heal jingle on Pokemon Center entry, music restored after onboarding dialog
- `src/game/systems/MenuSystem.ts` — `playSFX('menu-select')` on arrow navigation and selection
- `src/game/systems/DialogSystem.ts` — `playTextBlip()` (throttled 100ms) on each dialog character advance
- `src/game/systems/BattleSystem.ts` — `playBattleEventSFX()` for DAMAGE (with effectiveness routing), IMMUNE, FAINT, EXP_GAIN, LEVEL_UP, CATCH_ATTEMPT (shakes), CATCH_RESULT

### Tests
- `src/game/__tests__/AudioManager.test.ts` — 13 tests: singleton behavior, no-op safety, first-gesture, throttling
- `src/game/__tests__/audio-cry-params.test.ts` — 13 tests: species coverage, parameter validity, FM synthesis output
- `src/game/__tests__/chiptune-tracks.test.ts` — 12 tests: track count, BPM, loop flags, note structure, looping track length
